== Description ==
PEX for INDG is a wordpress plugin for easy assimilation of INDG's PEX URL into your wordpress website and render quick localization of strings to display information in any desired regional language.

= Key features =
* ShortCode Generation and associated replacement
* Localization of Strings
* Categorization of PEX

= Plugin website =
http://indg.com

== Installation ==
= From your WordPress dashboard =

1. Visit 'Plugins > Add New'.
2. Search for 'pex_for_indg'.
3. Activate from your Plugins page.

= Once Activated =
1. Visit 'Experiences > Add New' and Enter the PEX URL provided by INDG and Publish
2. Enter the Variant for the PEX
3. Modify (optional) the localization strings 
4. Update the post 
5. Copy the shortcode into your post. Done!

== Frequently Asked Questions ==
= What is the use of Shortcode? =

It’s a simple way for embedding PEX into your site content. Just Publish and Update your PEX to generate shortcode and copy&paste to the post content.

= Where I can find plugin documentation? =

Please visit http://indg.com
